URL Shortener Microservice

This microservice allows users to pass a URL as a parameter and receive a shortened URL

When you use the shortened URL, it will link to the original url

Example usage

https://short-url7.herokuapp.com/https://www.google.com

https://short-url7.herokuapp.com/https://www.yahoo.com

Will output

{"Long URL":"https://www.google.com","Short URL":1000}

{"Long URL":"https://www.yahoo.com","Short URL":2000}

Following usage

https://short-url7.herokuapp.com/1000

Will link to

https://www.google.com'
